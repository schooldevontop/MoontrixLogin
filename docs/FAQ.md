# MoontrixLogin FAQ

Q: Does it require other plugins?
A: No.

Q: Can I use SQLite?
A: Yes, default is SQLite.

Q: Which server platforms are supported?
A: Bukkit, Spigot, Paper, Purpur, Pufferfish, and Folia.
   Paper/Purpur/Folia are runtime-smoke-tested in CI (Folia runtime job currently uses 1.20.1).

Q: Which Minecraft versions are supported?
A: 1.8 to 1.21.11 for Bukkit/Spigot/Paper/Purpur/Pufferfish.
   Folia support is runtime-smoke-tested at 1.20.1 and compile-profile verified in CI.

Q: Do I need Java 21?
A: No. Current build target is Java 8 bytecode, so Java 8+ runtime is supported.

Q: Can I disable anti-bot?
A: You can raise thresholds, but disabling is not recommended.
